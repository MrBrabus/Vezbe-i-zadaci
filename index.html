Logički operator ! NOT

Logički operator ! NOT

U uslovima se koristi uz pomoć uzvičnika ! koji se stavlja ispred uslova.
Ovaj logički operator postavlja vrednost uslova suprotnu od one koja jeste.

! true = > false
! false = > true


Primer:

if (! 3>4){
alert("Zbog NOT uslov je ispunjen!")
}


Primer 2:

if (! (3>5 && 3>2) {
alert("Uslov je ispunjen!")
}

While petlja

U programiranju često imamo potrebu da neku akciju ili deo koda ponavljamo više puta.
While petlja nam omogućava da ponavljamo određene akcije sve dok je neki uslov ispunjen , bez njihovog ponovnog pisanja.

Sintaksa while petlje :

while ( uslov) {
blok koda koji se izvršava ukoliko je uslov true
}

Primer:

let i = 2;
while ( i>0) {
console.log(i);
i--;
}

Da bi se osigurali da je naša petlja nije beskonačna potrebno je da postoji situacija kada uslov nije ispunjen.

Svaki put kada se petlja izvrši , i smanjujemo za jedan. Nakon dva što se petlja dva puta izvršila uslov više nije ispunjen i ne izvršava se dalje.

Do while petlja

JavaScript poseduje i "do while" petlju koja nam omogućava da izvršimo neku instrukciju , i da je
ponavljamo sve dok je neki uslov ispunjen.

Razlika između while i do while petlje je u tome što će while petlja početi da izvršava neke
instrukcije ako je uslov ispunjen , dok će do while petlja izvršiti instrukciju jednom i ako je uslov
ispunjen nastaviće njeno izvršavanje.

Primer:

let i = 3;

    do {
   alert(i);
    i--;
}
    while(i>0);

U primeru iznad ispisaće se 3 , pa će se proveravati uslov i zatim će se ispisati 2 , 1 .

For petlja

For petlja je složenija od while petlje , ali isto tako veoma često primenjivana u programiranju.
Služi za ponavljanje nekih instrukcija onoliko puta koliko je određeno u definisanju for petlje.

Sintaksa for petlje:

for ( početna promenljiva , uslov , korak promene promenljive){
    // JavaScript instrukcije koje se izvršavaju
}

Primer:

for ( let i=0 ; i<10 ; i++){
    console.log("For petlja");
}

U primeru iznad vidimo for petlju uz pomoć koje će se ispisati deset puta "For Petlja". Iz koraka promene vidimo da će se i svaki put uvećati za 1 nakon izvršenja jednog ciklusa petlje. Nakon što se promenljiva poveća deset puta i neće biti veće od 10 i završava se izvršavanje petlje.
